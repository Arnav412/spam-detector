📧 AI Spam Detector

About This Project

  • This website uses Machine Learning to analyze text messages and classify them as Spam or Ham (Not Spam).

  • The model is trained on thousands of real SMS messages using CountVectorizer and Naive Bayes Algorithm.

Features

  • Detects Spam Messages
  • Machine Learning Based Prediction
  • Fast and Accurate Results
  • Simple and Interactive UI
  • Built using Flask and Python
